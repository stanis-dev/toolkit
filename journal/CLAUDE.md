# Journal

This project is meant for my note taking, journaling and long running deep research efforts

- Root directory is meant only for temporary artifacts, those must eventually be either moved into the folder for its type or removed once no longer useful.
- When working with research or analysis files, always read them in full -- even if they exceed single-read limits. Read in chunks until you have the complete content. Never skip sections or summarize from partial reads. Thoroughness is non-negotiable for research tasks.
- On EVERY main research file update you must keep in mind that new information can potentially cascade through the entire document. Always evaluate this. At the same time you must be mindful of what would be a simple update vs what would require an in-depth re-evaluation of strategy. If later, recommend a deep research to resolve it.
