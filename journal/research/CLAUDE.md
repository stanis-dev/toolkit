# Research

This folder is used for long running deep research efforts in order to allow for progressive expansion of a given topic, context preservation and generation of context-enriched prompts.

## Structure

- Each research has a main `research.md` file that contains the consolidated version of the research
- Deeper context lives in `/context` dir, e.g. deep research result files

## New research project

When creating a new research topic, create:
```
{topic-name}/
├── research.md    # Consolidated research (start with title + ## Summary + ## Key findings)
└── context/       # Deep research result files
```

## Rules

- File naming should be adjusted to improve readability
